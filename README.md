## Auth API (Go) - Mini Project

### Описание проекта

Это учебный мини-проект на Go, демострирующий основы тестирования HTTP API.

Реализован простой REST API с эндпоинтом /register, который
принимает email и пароль, валидирует входные данные и возвращает соотвествующий HTTP-статус.

Проект создан для практики:

- HTTP

- REST API

- unit-тестов

- тетсирования оброботчиков (handlers)

-------------------------------------------------
### Функциональноcть

Эндпоинт

`POST /register`

Тело запроса (JSON)
```json
{

"email":"test@test.com",

"password":"123456"

}
```
-----------------------------------------------
### Правила валидации

- email не должен быть пустым

- email должен содержать символ @

- пароль не должен быть пустым

- длина пароля должна быть не менее 6 символов

- поддерживается только метод POST
-----------------------------
### Ответы сервера
#### Успешная регистрация

201 Created
```json
{

"status": "registered"

}
```
-----------------------------------
### Ошибки
```md
Сценарий                 HTTP статус
Невалидный JSON	         400 Bad Request
Пустой email или пароль	 400 Bad Request
Пароль слишком короткий	 400 Bad Request
Неверный HTTP метод	 405 Method Not Allowed
```
----------------------------------------
### Тестирование

В проекте реализованы unit-тесты для HTTP handler'а.

Покрыты сценарии:

- успешный POST запрос

- некорректный JSON

- короткий пароль

- пустые поля

- запрос с неверным HTTP методом
